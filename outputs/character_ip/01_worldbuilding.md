# 角色IP設定書：我設計你 不是要你這麼難過耶

## 一、世界觀設定

### 1. 故事背景
在一個色彩繽紛、情感具象化的現代都市「情緒之都」中，每個人的情緒都會化為可見的光暈。這裡的人們習慣用情緒光暈來表達自己，並且有專門的「情緒設計師」協助大家調整、設計自己的情緒外觀。

### 2. 世界規則
- 情緒光暈會隨著個人心情變化而改變顏色與形狀。
- 情緒設計師擁有調整他人情緒光暈的特殊能力。
- 城市中有情緒能量場，過度負面情緒會導致城市局部失色。
- 積極正面的情緒能量可以修復、點亮城市。

### 3. 主要地標
- 情緒設計師學院：培養專業情緒設計師的學校。
- 光暈廣場：市民交流、展示情緒光暈的場所。
- 失色區：因負面情緒過盛而變得灰暗的區域。

### 4. 世界觀主題
本作主題為「理解與設計情緒」，強調每個人都能成為自己情緒的設計師，並鼓勵正向表達與自我療癒。
