# 一條龍 AI 代理服務流程設計

> 本文件詳述從客戶選購到升級的完整自動化服務流程，包含各階段對話腳本與退款政策，適用於 SaaS 型 AI 代理服務提供商。

---

## 1. 流程總覽

本服務流程共分為七個階段，形成一條龍自動化閉環：

```
客戶選購 → AI評估 → 報價 → 付款 → 安裝 → 客服 → 升級
```

每個階段皆由 AI Agent 自動觸發與執行，僅在特殊狀況下轉接真人客服。

---

## 2. 第一階段：客戶選購

### 2.1 觸發條件
客戶進入服務官網或透過 LINE/WhatsApp 等即時通訊工具發送「我想了解AI代理服務」、「我要購買方案」等關鍵字。

### 2.2 AI 對話腳本

```
AI：您好！歡迎來到 AI 代理服務中心。我是您的智能顧問小幫手。
AI：請問您目前最希望 AI 代理幫您解決哪一類問題？
    1. 客戶服務自動化（客服機器人）
    2. 銷售跟進與報價自動化
    3. 數據分析與報表生成
    4. 內部流程自動化（請假、審批等）
    5. 不確定，請幫我推薦

（客戶選擇後，AI 進一步引導）

AI：太好了！您選擇【{選項}】。請問您的企業規模大約是？
    1. 1-10 人（小型工作室）
    2. 11-50 人（成長型企業）
    3. 51-200 人（中型企業）
    4. 200 人以上（大型企業）

AI：再請教一個問題：您希望立即導入，還是先進行概念驗證（POC）？
```

### 2.3 資料收集
AI 自動建立客戶檔案，包含：
- 聯絡人姓名、Email、公司名稱
- 選購類別
- 企業規模
- 導入時間偏好
- 備註欄（客戶額外需求）

---

## 3. 第二階段：AI評估

### 3.1 自動化評估模型
根據客戶選項與規模，AI 評估引擎自動匹配以下維度：

| 評估維度 | 權重 | 評分方式 |
|-----------|------|----------|
| 需求匹配度 | 30% | 客戶選項與既有模組比對 |
| 技術成熟度 | 25% | 企業規模×技術能力預估 |
| 導入時程 | 20% | 概念驗證/立即導入 |
| 預算範圍 | 15% | 自動推算合理預算區間 |
| 擴充潛力 | 10% | 未來升級空間評估 |

### 3.2 評估結果輸出
評估完成後，AI 自動生成《導入建議書》，包含：
- 推薦方案名稱
- 功能清單與規格
- 預估導入時程
- 預估年費/月費
- 成功案例參考

### 3.3 評估階段對話腳本
```
AI：感謝您的耐心填寫！我正在為您進行智能評估…（約 15 秒）
AI：評估完成！根據您的需求與規模，我推薦最適合的方案——【專業客服方案 Pro】。
AI：這個方案包含：
    ✔ 24/7 智能客服機器人
    ✔ 支援 8 國語言
    ✔ 串接 LINE、官網、Facebook Messenger
    ✔ 每月 5,000 次對話額度
    ✔ 後台分析儀表板
AI：預估導入時間為 7-14 個工作天。
AI：請問您是否要立即查看詳細報價？
```

---

## 4. 第三階段：報價

### 4.1 動態定價機制
AI 依據評估結果自動產生客製化報價單，內含：

- **基礎月費**：依方案而定（如：NT$9,900 / 月）
- **一次性設定費**：首次導入設定費（如：NT$15,000）
- **超量費用**：超出月額度後每千次對話 NT$500
- **年度優惠**：年繳享 8 折優惠
- **加購項目**：API 串接加購、專屬模型訓練加購

### 4.2 報價單格式
AI 自動生成 PDF 報價單，發送至客戶 Email，同時在對話視窗顯示摘要。

### 4.3 報價階段對話腳本
```
AI：以下是您的專屬報價摘要：
    📌 方案：專業客服方案 Pro
    📌 月費：NT$9,900（年繳優惠：NT$95,040，平均每月 NT$7,920）
    📌 設定費：NT$15,000（僅首次）
    📌 超量費用：NT$500 / 千次

AI：詳細報價單已發送至您提供的 Email。
AI：請問您要：
    1. 立即付款，開始導入
    2. 稍後再說，寄送提醒
    3. 對報價有疑問，轉接專人
```

## 5. 第四階段：付款

### 5.1 付款方式
- 信用卡（一次性 / 每月自動扣款）
- 銀行轉帳
- 企業發票月結（限評分達 A 級以上客戶）

### 5.2 自動化付款流程
```
客戶選擇付款方式 → AI 生成付款連結/帳戶資訊 →
付款成功 → AI 自動發送收款確認與發票 →
系統自動開通權限 → 進入安裝排程
```

### 5.3 付款階段對話腳本
```
AI：感謝您的選擇！請點擊以下連結進行付款：
    🔗 [立即付款]（安全加密連線）

（付款成功後）
AI：✅ 付款成功！非常感謝您的信任！
AI：您的訂單編號為 #AIS-2025-0001
AI：發票將於 24 小時內寄送至您的 Email。
AI：接下來，我將為您安排系統安裝與設定。
```

## 6. 第五階段：安裝

### 6.1 安裝流程

| 步驟 | 動作 | 負責單位 | 預計時間 |
|------|------|----------|----------|
| 1 | 建立客戶專屬環境 | AI 自動 | 即時 |
| 2 | 發送設定指引 Email | AI 自動 | 付款後 5 分鐘 |
| 3 | 串接客戶指定渠道（LINE/Web/FB） | 工程團隊 + AI | 3-5 工作天 |
| 4 | 知識庫建置（FAQ、產品資訊等） | AI 自動匯入 | 即時 |
| 5 | 測試腳本驗證 | AI 自動測試 | 1 工作天 |
| 6 | 正式上線通知 | AI 自動 | 測試通過後 |

### 6.2 安裝階段對話腳本
```
AI：歡迎進入安裝階段！以下是接下來會發生的事：
    📋 步驟 1/6：我已為您建立專屬環境。
    📋 步驟 2/6：設定指引已寄出，請查看 Email。
    📋 後續步驟將由我們的系統與團隊協作完成。

AI：安裝期間如有緊急問題，請輸入「急件」轉接專人。
AI：您也可以隨時輸入「進度」查看當前安裝狀態。
```

## 7. 第六階段：客服

### 7.1 多層級客服架構

```
第一層：AI 客服機器人（24h 自動回應，解決率 > 85%）
第二層：專家 AI Agent（進階問題，如 API 串接、客製化需求）
第三層：真人客服（週一至週五 09:00-18:00）
```

### 7.2 常見問題與自動回應

| 問題類別 | AI 回應策略 |
|----------|-------------|
| 操作問題 | 提供圖文步驟教學與影片連結 |
| 帳務問題 | 查詢訂單狀態、提供歷史發票 |
| 技術問題 | 收集錯誤資訊→提供解決方案或轉接第二層 |
| 客訴問題 | 道歉+收集細節+轉真人客服 |

### 7.3 客服階段對話腳本
```
客戶：我串接 LINE 時出現錯誤訊息。

AI：很抱歉造成您的困擾！請提供錯誤代碼或畫面截圖，我來為您診斷。

（客戶提供後）

AI：已辨識到問題！這通常與 Webhook URL 設定有關。
AI：請按照以下步驟修正：
    1. 前往 LINE Developers Console
    2. 確認 Webhook URL 為：https://api.yourservice.com/webhook/line
    3. 點選「Verify」按鈕

AI：修正後請輸入「已修正」，我會為您進行連線測試。

（客戶表示問題已解決）

AI：太好了！連線測試通過 ✅
AI：請問還有其他問題需要協助嗎？
```

## 8. 第七階段：升級

### 8.1 升級觸發條件
- **使用量超過方案額度 80%** → AI 提醒升級
- **客戶主動要求** → 立即啟動升級評估
- **定期回訪** → 每季 AI 主動提供升級方案建議
- **新功能上線** → AI 自動發送升級通知與試用連結

### 8.2 升級流程
```
客戶同意升級 → AI 重新評估 → 生成新報價 →
補差價/重新付款 → 系統自動調整權限 →
升級完成通知
```

### 8.3 升級階段對話腳本
```
AI：系統偵測到您近三個月的使用量已達目前方案的 90%。
AI：為了確保服務順暢，我為您整理了一份升級建議：

    目前方案：專業客服方案 Pro（5,000次/月）
    建議升級：企業旗艦方案（20,000次/月 + 進階分析）
    升級月費：NT$24,900（年繳優惠 NT$19,920/月）
    可獲得：✔ 4倍額度 ✔ 進階數據分析 ✔ 專屬客戶成功經理

AI：升級後原有設定與歷史資料完全保留，無需重新安裝。
AI：請問您要立即升級嗎？
```

## 9. 退款政策

### 9.1 退款條件

| 情境 | 退款比例 | 備註 |
|------|----------|------|
| 購買後 7 天內未完成安裝 | 100% 全額退款 | 自付款日起算 |
| 購買後 14 天內，服務正常但不滿意 | 70% 退款 | 扣 30% 設定費 |
| 服務中斷超過 48 小時（非客戶端因素） | 按比例退還中斷天數費用 | 需經查證 |
| 超出方案額度產生超量費用爭議 | 退還超量費用 50% | 僅適用首次超量 |
| 洩漏客戶隱私資料 | 100% 全額退款 + 法律責任 | 經資安調查確認 |

### 9.2 退款流程（自動化）

```
客戶提出退款申請 → AI 驗證退款資格 →
自動計算退款金額 → 發送退款確認 Email →
系統自動執行退款（信用卡刷退/銀行轉帳）→
關閉服務權限（退款完成後 24 小時）→
滿意度調查
```

### 9.3 退款對話腳本
```
客戶：我想申請退款。

AI：了解，請問您退款的原因是什麼？
    1. 不符合預期
    2. 技術問題無法解決
    3. 預算考量
    4. 其他原因

（客戶選擇後）

AI：感謝您的回饋。我來為您查詢退款資格…
AI：您的訂單符合【7天全額退款】條件，可退還 NT$XX。
AI：退款將在 5-7 個工作天內退回原付款方式。
AI：退款完成後服務將於 24 小時內關閉，請提前備份所需資料。
AI：請問要立即執行退款嗎？

（客戶確認後）

AI：✅ 退款申請已提交！您將收到退款確認 Email。
AI：我們非常重視您的意見，您的回饋將幫助我們持續改善。
```

### 9.4 退款注意事項
- 年繳客戶退款按「已使用月份比例」計算，並取消優惠折扣
- 設定費若已執行串接作業，恕不退還
- 退款爭議無法透過 AI 解決時，自動轉接退款專員
