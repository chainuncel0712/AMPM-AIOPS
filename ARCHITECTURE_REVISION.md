```
┌─────────────────────────────────────────────────────────────────────────────┐

│  │ 決策層       │ 路由權限     │ KERNEL   │ 模型選擇、提供商平衡、    │     │

│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 認知層       │ 推理權限     │ KERNEL   │ 推理、反思、評論、規劃    │     │
│  │              │              │          │ 不可修改路由或治理規則    │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 演化層       │ 適應權限     │ KERNEL   │ 評分權重、優化提案        │     │

│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 記憶子系統   │ 儲存權限     │ KERNEL   │ 寫入策略、保留期、索引    │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 治理安全     │ 執行權限     │ KERNEL   │ 權限引擎、沙箱規則、      │     │
│  │              │              │          │ 審計記錄、防火牆          │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 執行時       │ 執行權限     │ KERNEL   │ 上下文組裝、工具分發、    │     │
│  │              │              │          │ 狀態管理                  │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 生命週期     │ 管理權限     │ KERNEL   │ 代理身分、人格配置、      │     │
│  │              │              │          │ 資源回收                  │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 企業系統     │ 營運權限     │ PUBLIC   │ 帳單、授權、部署、租戶    │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ 插件         │ 能力權限     │ PUBLIC   │ ✅ 執行工具               │     │
│  │              │              │          │ ✅ 回傳結構化輸出          │     │
│  │              │              │          │ ✅ 提供上下文候選          │     │
│  │              │              │          │ ❌ 修改路由               │     │
│  │              │              │          │ ❌ 修改治理               │     │
│  │              │              │          │ ❌ 修改權限               │     │
│  │              │              │          │ ❌ 修改編排流程           │     │
│  ├──────────────┼──────────────┼──────────┼──────────────────────────┤     │
│  │ SDK          │ 介面權限     │ PUBLIC   │ API 合約、開發工具        │     │
│  └──────────────┴──────────────┴──────────┴──────────────────────────┘     │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  權限隔離規則:                                                              │
│  ┌────────────────────────────────────────────────────────────────────┐     │
│  │ 1. 僅 KERNEL 擁有路由、治理、上下文、演化權限                     │     │
│  │ 2. PUBLIC 生態層不得持有任何決策權限                              │     │
│  │ 3. 插件僅是能力提供者，不擁有任何權限                            │     │
│  │ 4. 企業系統在執行時核心外部運作                                  │     │
│  │ 5. 演化只能提案，不可自動套用 — 所有變異需治理批准                │     │
│  │ 6. 所有跨層操作必須經過治理層                                    │     │
│  │ 7. EventLog 記錄每次 decision → tool_call → output，支援完整重播 │     │
│  └────────────────────────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────────────────────────┘
```
