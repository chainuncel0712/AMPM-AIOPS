# AI‑BOS Core Architecture

## Overview

AI‑BOS is a **biological operating system** for AI agents. Instead of pipelines or graphs, it organizes capabilities as **organs** within a living body.

## Layers

1. **Core** — Kernel, Config, Context
2. **Organs** — Brain, Memory, Tools, Registry
3. **Lifecycle** — Boot, Baseline, Health, Shutdown
4. **Executor** — Orchestrator, Reflect, Repair, Evolve, Tools

## Design Principles

- **Modularity** — Each organ is independent and replaceable
- **Lifecycle** — Every agent follows a predictable lifecycle
- **Safety** — Governance and immune layers in commercial edition
- **Evolution** — Self-improvement through reflection cycles
