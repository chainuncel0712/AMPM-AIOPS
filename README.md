<h3>🔧 本地開發</h3>

```bash
# 1. 複製倉庫
git clone https://github.com/chainuncel0712/AMPM-AIOPS.git
cd AMPM-AIOPS

# 2. 設定環境變數
cp .env.example .env
# 編輯 .env 填入你的 API Keys

# 3. 安裝依賴
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 4. 啟動
python3 main.py
```

<h3>🐳 Docker 部署</h3>

```bash
docker build -t ampm-aiops .
docker run -d \
  --name ampm-aiops \
  --env-file .env \
  -p 5050:5050 \
  ampm-aiops
```
