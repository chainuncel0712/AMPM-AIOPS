<h2 align="center">🔗 與 AMPM-KEL 整合</h2>

公開組件嚴格通過明確定義的介面與私有內核互動：

| 介面 | Interface | 用途 |
|:---|:---|---:|
| 🧩 插件介面 | Plugin Interface | 插件註冊、能力暴露、結果回傳 |
| 📦 SDK 介面 | SDK Interface | 查詢、提交、訂閱 |
| 📡 事件總線 | Event Bus | 非同步事件發布/訂閱 |
| 🔄 生命週期 | Lifecycle Interface | 啟動/停止/健康檢查 |

> ⚠️ **嚴格禁止**直接導入 AMPM-KEL 內部組件。
